acronym - type
( 1 ) regular acronym - type abbreviations : such character in the abbreviation is contained in the definition ( e. g., < DC, dendritic call *)
( Doome irregular acronym - type ones : only one character in the abbreviation is not contained in the definition ( e. g., < CNS1, cyclophilin seven suppressor *)
non - acronym - type mainly several characters in the abbreviation are not contained in the definition ( e. g., < Fas, CD95 >, < 5 - HT, serotonin >, < Pax6, eyeless >)
